CougarWatch Antivirus

CougarWatch is the name of a project that I and four others worked on together. We created a functional antivirus featuring both heuristic and hash-based detection, functional user sign-in, and an interactive gui.

Password Checker

This is a password checker. It compares your input to a list of known common passwords, then analyzes the password based on length, variety of characters, etc. It outputs a score based on your password's strength.
